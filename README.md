# 控制點查詢系統(英文檔名版)

## 快速啟動

雙擊 `啟動網站.bat`,或在此資料夾開命令視窗執行:

    cd web
    python serve.py

然後瀏覽器開 http://localhost:5000

成功時命令視窗會印出:
    * 載入 9915 點 | 67圖根(全):4352 | ...

## QGIS 工作流程

在 `layers/current/` 內用 QGIS 編輯這些圖層,存檔後網站 30 秒內自動更新。

## 讓同事連線

1. 命令視窗打 ipconfig,找「IPv4 位址」(例 192.168.1.50)
2. 同事瀏覽器開 http://192.168.1.50:5000
3. Windows 防火牆需放行 TCP 5000 埠
